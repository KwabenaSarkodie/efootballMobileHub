## 🌟 About The Project

Welcome to **eFootball HUB**, the premier destination for eFootball Mobile enthusiasts! Designed with a sleek, mobile-first interface, this platform empowers players to dominate the pitch and master the meta. Whether you're looking for the best player builds, want to design your own custom cards, or need to assemble your unbeatable squad, eFootball HUB has you covered.

### ✨ Key Features

- 🔍 **Player Database & Search**: Find max stats, discover hidden gems, and locate epic legends instantly.
- 🎨 **Card Creator**: Design your own premium custom cards instantly with real-time preview (Epic, POTW, Showtime, etc.).
- 🏟️ **Squad Builder**: Create your ultimate 11 on a visual pitch and test your team chemistry.
- ⚖️ **Player Comparison**: Compare any two players side-by-side to make the smartest transfer decisions.
- ⚙️ **Custom Builds Simulator**: Allocate progression points and simulate max ratings before committing in-game.

## 🚀 Getting Started

This project is built using pure vanilla **HTML, CSS, and JavaScript**. No heavy frameworks, just clean, fast, and responsive code.
